Understanding CSS Positions

In CSS, the position property decides how an element sits on the page. Some elements stay in the normal flow, some move a little from their usual spot, and some can stick to the screen or a parent box. These pages explain each one in simple words with small visual examples.

Static

The normal default position. Elements stay where they naturally belong.

Relative

The element keeps its space but can move a little from its original spot.

Absolute

The element can be placed exactly where you want inside a nearby parent.

Fixed

The element stays attached to the screen even when the page scrolls.

Sticky

The element behaves normally at first, then sticks while you scroll.
