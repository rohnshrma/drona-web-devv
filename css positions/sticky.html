Position: sticky

Sticky is like a mix of relative and fixed. At first, the element acts like a normal element in the page. But when you scroll and it reaches a chosen point, it sticks there instead of disappearing. This is useful for section headings or menus that should stay visible while you read.

I become sticky when you scroll

Sticky elements are helpful because they do not start fixed immediately.

They first behave like normal content, which feels natural in the layout.

Then they stick only when needed during scrolling.

This makes them great for labels, headings, or navigation areas.
